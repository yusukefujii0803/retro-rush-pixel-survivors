# RETRO★RUSH : Pixel Survivors

レトロな8ビット風グラフィックとチップチューン音楽で彩られたアクションゲーム

## ゲーム概要

- **戦略的アクション**: ジャンプで敵の攻撃を回避し、射撃で敵を倒そう
- **ボス戦システム**: 15体撃破でボス出現、ボスを倒してゲームクリア
- **手動射撃**: スペースキーで狙いを定めて射撃
- **多様な敵**: 基本、強敵、高速、空中敵の4タイプ
- **爆発回避**: ボスの地面爆発攻撃をジャンプで回避

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 操作方法

- **クリック/タップ**: ジャンプ（爆発を回避！）
- **スペースキー**: 射撃
- **右クリック**: 射撃（代替）

## 技術スタック

- **ゲームエンジン**: Phaser 3
- **ビルドツール**: Vite
- **開発言語**: JavaScript (ES6+)
